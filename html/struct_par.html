Descrição detalhada

Sistema para gerenciar memória virtual

Esse programa tem como objetivo traduzir dos lógicos para endereços físicos para um espaço de endereço virtual de tamanho 2 16 = 65.536 bytes. Ele lê o arquivo address.txt que tem varios endereços lógicos e, utilizando um TLB e uma uma tabela de páginas, ele vai traduzir cada um dos endereços lógicos para seu endereço físico correspondente, e também o valor do byte armazenado no físico traduzido Criando a struct Par para construirmos a TLB
